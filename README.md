# GET_Group
Registration Selenium Task using Java, Maven, TestNG, POM, Properties File, Cucumber

At first, the user navigates to the URL (https://codenboxautomationlab.com/registration-form/) to get register for Training courses
We assert that the first letter in First Name and Last Name is a capital letter and First Name and Last Name aren't the same

 We use the Faker library to generate unique test data for (first name, last name, email, phone)
 Also, we take screenshots of failed test cases and we use BDD Cucumber


If you want to run the feature
After downloading the project run "RunnerTest" class from "runner" package
